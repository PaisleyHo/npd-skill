# Changelog

中文说明见下半部分。

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, adapted for a skill-template repository.

## [0.1.0] - 2026-04-07

### Added

- Initial `npd-skill` release
- English and Chinese skill entry files
- Persona, memory, and corrections workflow
- Chinese intake template
- Filled intake example
- Fictional sample profile
- Merge rules for incremental updates
- Version history file
- English and Chinese examples
- English and Chinese scenarios
- English and Chinese persona reference files
- Repository `README.md`
- `LICENSE` under MIT
- Chinese project homepage README via `README.zh-CN.md`

### Notes

- This release is designed as a reusable mother-template for future skills such as `bpd-skill`, `ocpd-skill`, and related persona-state variants.
- The workflow is intentionally modeled after an `ex-skill`-style layering approach: persona + memory + corrections.
